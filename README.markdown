# Pencil
Simple jQuery HTML5 WYSIWYG

## Author
Ilya Shalyapin, ishalyapin@gmail.com

## Features
AJAX image uploading

##Browser compatibility
Firefox, Opera, Chome.

## Requirements
 - jQuery (http://jquery.com/)
 - JQuery Form Plugin (http://malsup.com/jquery/form/)
